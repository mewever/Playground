# Playground
This is a repository I use for trying out new ideas or just practicing an implemenation of something I haven't tried before or haven't done in a particular language.

This is primarily intended as an example of my coding skills and style, but this code is generally available for you to copy. I would ask that if you find something here useful enough to use in your own project, that you let me know about it.
I'm always interested to see what others are doing and would love to know how my code was useful to you.

# Projects
## LinkedList
This is an implementation of linked lists in C#. It's not functionally any better than the List<> that already exists, but was just me looking into how to build a linked list from scratch in C#.

## SignalRAuction
This is an example of a client for a live online auction built with SignalR. The interface is more functional than pretty, but includes real-time updates on bids and removing items from the list as they close.

Most of the UI is pure JavaScript, so this provides a good example of my coding approach with that language.
